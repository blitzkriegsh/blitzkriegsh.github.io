Part 1 : Coding
Coding and Debugging is an event where the participants are welcome to code in any language and utilize any features provided by the standard library of the said language. The difficulty of rounds increases as rounds proceed.
Part 2 : Capture the flag
This is an event that aims to bring out the creative minds of the participants with the help of technology. This event is conducted in two rounds. The first round is of less difficulty level which consists of OSINT challenges, ciphers, algorithms, etc. The second round is of medium difficulty in which the question covers other various categories of CTF challenges. Observe, Decipher and Resolve are the keys to Capture the Flag.

Hackathon Rules

  • Participants are to participate in groups of two.
  • Participants are welcome to code in any language and utilize any features provided by the standard library of the said language.
  • Mobile devices and other electronic gadgets are prohibited during the competition.
  • Knowledge of reading data from the file system using the selected programming language is the minimum expected requirement.
  • Participants will be provided with systems with vscode, dev cpp, python-idle, and notepad-c++. Requests for any additional software should be relayed before 11th of February, 2024, via the student coordinator.
  • The use of AI-tools like Chat-GPT should be as per the instructions from the coordinators.
  • The hackathon will be conducted in a round-to-round basis spanning multiple rounds, tallying the scores in each round.
  • The CTF round is categorized into 2: Easy and Medium levels.
  • The participants must be familiar with the file formats, Ciphers and algorithms.
  • Both the rounds covers various categories such as OSINT, Steganography, Cryptography, Cyber forensics
  • Moderators reserve the right to interpret and enforce these rules.
  • Participants must be capable of explaining their code when required by an Examiner.
  • The FINAL decision will be taken by the examiner.
  • VIOLATION of these rules will not be tolerated and will be rewarded with DISQUALIFICATION.
