Round 1 : Coding and Debugging
Coding and Debugging is an event where the participants shall be tested on their proficiency in the C, C++, and Python programming languages. The event shall consist of 3 rounds. A preliminary round, to test the participants on their programming knowledge, through which the qualifying teams shall proceed to the coding round where the participants shall be required to code for the questions provided to them. The final round will require the participants to fix the errors and debug the code provided to them to obtain the output required.
Round 2 : Capture the flag (CTF)
This is an event that aims to bring out the creative minds of the participants with the help of technology. The Hackathon shall be solely based on the topic of Cybersecurity. This event is conducted in two rounds. First round is a hacking round with low difficulty, the second round is with medium difficulty which is conducted only if any tie occurs.

Hackathon Rules

  • Participants are to participate in groups of two.
  • Participants are welcome to code in any language and utilize any features provided by the standard library of the said language.
  • Mobile devices and other electronic gadgets are prohibited during the competition.
  • Knowledge of reading data from the file system using the selected programming language is the minimum expected requirement.
  • Participants will be provided with systems with vscode, dev cpp, python-idle, and notepad-c++. Requests for any additional software should be relayed before 11th of February, 2024, via the student coordinator.
  • We strictly prohibit the use of AI models such as Chat-GPT and the like.
  • The hackathon will be conducted in a round-to-round basis spanning multiple rounds, tallying the scores in each round.
  • Moderators reserve the right to interpret and enforce these rules.
  • Participants must be capable of explaining their code when required by an Examiner.
  • The FINAL decision will be taken by the examiner.
  • VIOLATION of these rules will not be tolerated and will be rewarded with DISQUALIFICATION.
