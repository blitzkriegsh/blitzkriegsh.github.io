Rules

  • Game Settings:
    • Match Length: 7 minutes per half.
    • Difficulty Level: Consistent across all matches (e.g., Professional, World Class).
    • Injuries: On or off, depending on preference.
    • Extra Time: Decide whether to use extra time and penalties for knockout rounds.
  • Team Selection:
    • Club Teams: Participants can choose any club team available in FIFA 23.
  • Controller Settings:
    • Default Settings: Use default FIFA 23 control settings.
    • Custom Controls: Allow participants to use custom controls if they prefer.
  • Disconnections and Technical Issues:
    • Reconnections: If a match disconnects due to technical issues, attempt to reconnect. If not possible, the score at the time of disconnection stands.
    • Console Issues: If a console malfunctions, attempt to switch to another console or use a backup console.
  • Code of Conduct:
    • Respect: All participants are expected to treat each other with respect.
    • Fair Play: Cheating, exploiting glitches, or using external devices is strictly prohibited.
    • Sportsmanship: Handshakes or virtual gestures of sportsmanship are encouraged before and after matches.
